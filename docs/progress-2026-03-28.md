# Progress Report - 2026-03-28

Ringkasan pekerjaan harian untuk Aegis Bridge dan AegisNexus.

## Aegis Bridge (CLI)

- Dibuat CLI auth Copilot di `copilot-cli.mjs`.
- Device Flow Login:
  - Request device code ke GitHub.
  - Polling OAuth access token.
  - Exchange ke Copilot session token.
- Penyimpanan token aman di `github-copilot.token.json`.
- Penambahan chat test ke endpoint Copilot.
- Penambahan mode daftar model + pemilihan model.
- Pembatasan model ke `gpt-5-mini` dan `gpt-4o`.

## AegisNexus (Web Orchestration)

- Dibangun backend TypeScript (`server.ts`) + frontend TypeScript (`frontend/app.ts`).
- Mode orchestration:
  - General mode (Queen direct reply).
  - Full orchestration (Queen + Planner + Worker).
  - Custom override (toggle planner/worker).
- Smart intent classification untuk routing general vs complex task.
- Execution indicator real-time via SSE log stream.
- Safety protocol:
  - Throttle acak 3-6 detik antar request.
  - Retry saat HTTP 429 dengan pause 30 detik.
  - Max iteration 3 dan pause menunggu approval.
- Token refresh otomatis jika sisa masa berlaku <= 30 menit.
- Memori percakapan SQLite (`aegisnexus.db`):
  - Tabel `sessions` dan `messages`.
  - Pagination history `ORDER BY timestamp DESC LIMIT ?` lalu dibalik ASC.
- Persona engine berbasis Markdown (`personas/the_queen.md`) dan selalu di-inject sebagai system prompt index 0.
